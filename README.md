# La Birria Real-Time POS

A real-time Point of Sale and Kitchen Display System inspired by the provided Birria poster theme.

## Features

- Poster-inspired UI theme (orange/yellow/green palette)
- Full menu with signatures, combos, drinks, and specials
- Role login for admin and cashier (cashier PIN login)
- Live cart, tax, totals, and cash change calculation
- Payment methods: cash, card, M-Pesa
- Direct receipt printing on successful checkout
- Admin dashboard with sales done list, menu management, and cashier/password settings

## Run

```bash
npm install
npm start
```

Open:

- `http://localhost:470`
